# GitHub 사용자명 확인 방법

## 방법 1: GitHub 웹사이트에서 확인

1. https://github.com 로그인
2. 우측 상단 프로필 아이콘 클릭
3. **"Signed in as USERNAME"** ← 여기가 사용자명

## 방법 2: 프로필 페이지 URL 확인

1. GitHub 로그인
2. 우측 상단 프로필 아이콘 → **Your profile** 클릭
3. 브라우저 주소창 확인:
   - `https://github.com/YOUR-USERNAME` ← 이 부분이 사용자명

## 방법 3: Settings에서 확인

1. GitHub → Settings
2. **Public profile** 섹션
3. **Username** 필드 확인

---

## 레포지토리 목록 확인

1. GitHub 로그인
2. 우측 상단 프로필 아이콘 → **Your repositories**
3. 클론하려는 레포지토리 클릭
4. 녹색 **Code** 버튼 클릭
5. HTTPS URL 복사
