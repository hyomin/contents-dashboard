# 플랫폼별 최적 영상 스펙 가이드 (2026)

유튜브 쇼츠, 인스타 릴스, 틱톡 최신 스펙 정리. **영상·스크립트·자막 설계 전에 반드시 적용.**

## 핵심 요약

- 세 플랫폼 공통: **1080×1920px, 9:16 세로** → 한 번 제작·세 곳 업로드 가능
- **통합 안전 영역:** 화면 중앙 **1080×1300px** — 자막·얼굴·로고·CTA는 이 안에만
- CapCut Export: 1080×1920, 9:16, H.264, AAC, 30fps, 비트레이트 8Mbps+

---

## 1. YouTube Shorts

| 항목 | 사양 |
|------|------|
| 해상도 | 1080×1920 (Full HD 세로) |
| 비율 | 9:16 필수 |
| 최대 길이 | 3분 (180초) |
| 알고리즘 최적 | **15~60초** (60초 이하 노출 유리) |
| 포맷 | MP4, H.264, AAC, 30/60fps, 8Mbps+ |
| 썸네일 | 1280×720 |

**주의**

- 가로(16:9) 업로드 → 쇼츠 분류 안 됨
- 720p 미만 픽셀화
- **60초 초과** 시 유료 음원 제약 → 무료/Suno BGM 권장
- **하단 약 20%**에 중요 정보·자막 금지 (폴더블·UI)

**안전 영역 (중앙):** 960×1350 참고

---

## 2. Instagram Reels

| 항목 | 사양 |
|------|------|
| 해상도 | 1080×1920, 9:16 |
| 최대 길이 | 3분 (일부 10분) |
| 알고리즘 최적 | **15~90초** (바이럴은 짧게) |
| 포맷 | MP4/MOV, H.264, AAC 128kbps+, 30fps, 4GB |

**4:5의 늪**

- 릴스 탭 9:16, **메인 피드 4:5 크롭** → 핵심은 중앙 **1080×1350**
- 자막: 하단 **250px 이상** 여백 (UI 가림)
- **틱톡 워터마크** 영상 업로드 시 노출 급감 → 제거 후 업로드

---

## 3. TikTok

| 항목 | 사양 |
|------|------|
| 해상도 | 1080×1920, 9:16 기본 |
| 알고리즘 최적 (세로) | **21~34초** |
| 포맷 | MP4/MOV, H.264, AAC, 30/60fps |

**우측 아이콘 부대**

- 우측 **150~180px** 비움 — 자막·텍스트 금지
- 안전 영역 (중앙): **880×1500** 참고

**참고:** 1분+ 가로 영상은 전략적 부스트 가능 (기본 숏폼 전략은 9:16 세로)

---

## 4. 세 플랫폼 비교 (요약)

| 항목 | YouTube Shorts | Instagram Reels | TikTok |
|------|----------------|-----------------|--------|
| 해상도 | 1080×1920 | 1080×1920 | 1080×1920 |
| 최적 길이 | 15~60초 | 15~90초 | 21~34초 (세로) |
| 통합 안전(중앙) | 960×1350 | 1080×1350 | 880×1500 |

---

## 5. 통합 안전 영역 (스크립트·편집 필수)

- **1080×1300px** 중앙 박스에 핵심 비주얼·자막 위치 가정
- 스크립트 총 길이: **YouTube·릴스 기준 45~60초**, TikTok 단독 타깃 시 **21~34초** 버전 고려 가능 (제작 메모에 표기)

---

## 6. 자주 하는 실수

| 실수 | 해결 |
|------|------|
| 가로 영상 | 9:16 세로 only |
| 자막 하단·우측 끝 | 통합 안전 영역·우측 180px 여백 |
| 720p export | 1080p |
| 60초+ 쇼츠 유료 BGM | 무료 음원/Suno |
| 릴스 4:5 미고려 | 중앙 1080×1350 |
| 틱톡 WM → 릴스 | WM 제거 |
