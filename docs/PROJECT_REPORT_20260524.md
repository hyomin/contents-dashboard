## 1. 개발 로드맵 우선순위 표

> 현재 구현 완료 기능을 기준으로, 다음에 개발해야 할 영역을 우선순위 순으로 정리합니다.

| 순위 | 영역 | 현재 상태 | 세부 작업 | 예상 공수 | 비고 |
|------|------|-----------|-----------|-----------|------|
| 🔴 **1** | **보안 강화** | API 키·비밀번호 `.env.local` 평문 노출 | `.env.local` 깃 추적 여부 확인, Supabase service_role·YouTube API Key·Notion 토큰 전 로테이션, `lang="ko"` 수정 | 0.5일 | **즉시 필요** — 키 유출 시 과금 발생 |
| 🔴 **2** | **TikTok 실수집** | UI만 존재, 더미 데이터 | Apify TikTok Scraper 연동, `videos` 테이블 플랫폼 통합, vs.Avg 계산 로직 | 3~5일 | Apify 유료 플랜 필요 |
| 🔴 **3** | **Instagram 실수집** | «준비 중» UI | Apify Instagram Scraper 연동, 릴스/피드 구분, 좋아요·댓글 지표 수집 | 3~5일 | TikTok과 병행 가능 |
| 🟠 **4** | **주제 선별 AI 복구** | archived (LangChain 패키지 필요) | n8n에 LangChain 패키지 설치, `N8N_TOPIC_SUGGEST.json` 재임포트, `/api/topic-suggest` 연동 활성화 | 1~2일 | Gemini Pro 도입 시 품질 대폭 향상 |
| 🟠 **5** | **수익 추적 완성** | 일부 «준비 중» | YouTube Analytics API 실수익 연동, AdSense 수익/RPM/CPM 표시, 영상별 수익 추정 | 3~4일 | YouTube Analytics API 별도 OAuth 필요 |
| 🟠 **6** | **n8n 워크플로 7개 클라우드 배포** | 로컬 Docker만 동작 | Vercel `vercel.json` cron 설정, n8n Cloud 또는 Railway 이전, `DASHBOARD_API_URL` 프로덕션 변경 | 1~2일 | 로컬 종료 시 자동화 중단 문제 해결 |
| 🟡 **7** | **기간별 추이 차트** | 없음 (현재 정적 테이블) | Recharts/Chart.js 도입, `videos` 날짜별 vs.Avg 변화 그래프, 플랫폼별 성장 추이 | 2~3일 | 데이터 축적이 선행되어야 유의미 |
| 🟡 **8** | **AI 콘텐츠 초안 생성** | `content-studio` 더미 | Gemini API로 롱폼 스크립트·썸네일 문구 자동 생성, 아웃라이어 기반 주제 연동 | 3~5일 | 4번(주제 선별) 완료 후 진행 |
| 🟡 **9** | **Shadcn/Radix UI 도입** | 커스텀 Tailwind만 사용 | Shadcn 컴포넌트 점진적 교체 (Button, Dialog, Table, Select), `nuqs` URL 상태 관리 | 3~4일 | 코드 규칙과 일치, UX 품질 향상 |
| 🟢 **10** | **테스트 코드 작성** | `verify:collect` 스크립트만 존재 | 수집 함수 단위 테스트 (Vitest), API 라우트 통합 테스트, Playwright E2E | 5~7일 | 안정성 확보 후 신규 기능 개발 가속 |
| 🟢 **11** | **CTR·시청지속률 지표** | 없음 | YouTube Analytics API 추가 지표 수집, 영상별 CTR/AVD 저장, Outlier 분석 연동 | 2~3일 | Analytics API OAuth 완료 후 |
| 🟢 **12** | **멀티 플랫폼 발행 자동화** | repurpose·deploy 화면만 존재 | n8n으로 유튜브 자막→블로그→인스타 리퍼포징 파이프라인, 캘린더 자동 등록 | 5~7일 | 8번(AI 생성) 완료 후 최종 단계 |

### 권장 실행 순서 (Phase)

```
Phase 0 (즉시)   : 보안 강화 → .env 키 로테이션
Phase A (1~2주)  : TikTok + Instagram 실수집 → 멀티플랫폼 분석 기반 완성
Phase B (3~4주)  : 주제 선별 AI 복구 + 수익 추적 + 클라우드 배포
Phase C (5~7주)  : 추이 차트 + AI 콘텐츠 생성 + Shadcn 도입
Phase D (8~12주) : 테스트 + CTR 지표 + 멀티플랫폼 자동 발행
```

## 2. Gemini Pro 구매 시 퍼포먼스 향상

> 현재 프로젝트는 **Gemini 1.5 Flash (무료)** 를 사용 중입니다.  
> 무료 한도: 일 1,500회 / 분 15회 요청, 컨텍스트 100만 토큰.  
> **Gemini 2.0 Pro / 1.5 Pro** 유료 업그레이드 시 기대 효과:

| 영역 | 현재 (무료 Flash) | Pro 업그레이드 후 | 향상 폭 | 적용 위치 |
|------|------------------|------------------|---------|-----------|
| **RSS 주제 AI 정제 품질** | 기본 제목 요약, 단순 이유 설명 | 맥락 이해·페르소나 반영 제목, 세밀한 이유 분석 | ★★★★☆ | `rss-topic-collect.ts` / `ai_title`, `ai_reason` |
| **RSS 처리 속도·한도** | 분당 15회 → 50개 피드 처리 시 병목 발생 | 분당 2,000회 (Tier 2+) → 병목 해소 | ★★★★★ | `runRssTopicCollect()` rate limit 제거 |
| **주제 선별 AI 응답 정확도** | Flash 기반 단순 분류 | Pro 다단계 추론 → 아웃라이어·트렌드 교차 분석 | ★★★★☆ | 4번 로드맵 `/api/topic-suggest` |
| **롱폼 스크립트 자동 생성** | 짧은 컨텍스트, 일관성 낮음 | 200만 토큰 컨텍스트 → 전체 채널 분석 후 스크립트 | ★★★★★ | 8번 로드맵 `content-studio` |
| **썸네일 문구·훅 생성** | 기본 카피 | 경쟁 채널 아웃라이어 패턴 학습 후 최적화 카피 | ★★★☆☆ | AI 인사이트 뷰 확장 |
| **일일 처리 가능 영상 분석 수** | ~1,500개/일 | 무제한 (과금 기반) → 대규모 경쟁 채널 분석 가능 | ★★★★★ | `analytics-from-videos.ts` 확장 |
| **다국어 콘텐츠 생성** | 한국어 중심, 영어 품질 낮음 | 100개 이상 언어 고품질 지원 | ★★☆☆☆ | 글로벌 채널 확장 시 |
| **Grounding (Google 검색 연동)** | 기본 지원 (무료 1,500회) | 실시간 검색 연동 AI 인사이트 무제한 | ★★★☆☆ | `insights` API AI 분석 |

**월 예상 비용:** Gemini 1.5 Pro ≈ $7/월 (100만 토큰 입력 기준), Gemini 2.0 Pro ≈ $10~20/월  
**ROI 관점:** RSS 피드 50개 일 2회 수집 × 30일 = 3,000회/월 → 무료 한도 초과 즉시 병목 해소 가능

## 4. Gemini vs Magnific — 구매 우선순위 비교

> **현재 상황:** `gemini-2.5-flash` 무료 티어 사용 중 (일 1,500회 한도, 분 15회 제한)  
> **두 도구는 용도가 다릅니다** — 택일이 아닌 순서 결정의 문제입니다.

### 4-1. 핵심 역할 비교

| 구분 | Gemini API (유료 티어) | Magnific Premium+ |
|------|----------------------|-------------------|
| **카테고리** | 텍스트 AI / 분석·생성 | 비주얼 AI / 이미지·영상·오디오 생성 |
| **현재 사용 여부** | ✅ 사용 중 (RSS 정제) | ❌ 미사용 |
| **대시보드 연동** | ✅ 직접 연동 (API Key 설정됨) | ❌ 자동화 불가 (Enterprise만 API) |
| **월 비용** | 사용량 기반 $5~30 (Flash → Pro) | 고정 $33.75/월 |
| **즉각 효과** | 현재 bottleneck(분당 15회) 즉시 해소 | 새 워크플로 구축 필요 |

### 4-4. 구매 권장 순서

```
▶ 지금 즉시 (비용 최소화)
  → Gemini 2.5 Flash 유료 전환 (~$5~8/월)
  → 분당 한도 해소 + RSS 처리 안정화

▶ Phase B (콘텐츠 기획 AI 구축 시)
  → Gemini 3.1 Pro 업그레이드 (~$20~50/월)
  → 스크립트 자동 생성 · 아웃라이어 패턴 분석 · 주제 선별 AI

▶ Phase C (채널 성장 · 영상 퀄리티 투자 시)
  → Magnific Premium+ ($33.75/월)
  → 썸네일 AI 생성 · 쇼츠 클립 · 업스케일링
```

> **결론:** 이 대시보드 프로젝트에는 **Gemini 먼저**가 맞습니다.  
> 대시보드 코드에 이미 연결되어 있고, 현재 병목이 존재하며, 자동화 파이프라인 전체에 효과가 있습니다.  
> Magnific은 채널 영상 제작량이 늘어나는 시점에 추가하면 투자 효율이 높습니다.

## 6. 리팩토링 완료 사항 (2026-05-24)
