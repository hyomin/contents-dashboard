### 레이아웃 분석

**1) pokara61 — 시황 분석형**
- 제목에 `<시장>` 같은 꺽쇠 태그로 글의 카테고리를 표시.
- 한두 문장마다 줄바꿈(가독성 위주, Smart Editor의 짧은 단락).
- `*` 한 줄을 구분선처럼 사용해 문단 블록을 분리.
- 본문 중간에 차트 이미지 삽입 + 이미지 캡션으로 설명("국제유가(WTI) 일봉차트: ...").
- "수혜주는?" 같은 소제목을 굵게 강조하되 별도 H태그 없이 본문 텍스트로 처리.
- 마지막에 최신 속보를 추가하는 추가 단락("14일 제네바 서명 임박") — 발행 후 업데이트하는 라이브 기사 형식.

**2) self_economy — 종목 리포트 해설형**
- 제목 자체가 본문 첫 줄에 그대로 다시 등장(타이틀 반복으로 SEO/공유 시 가독성 확보).
- "현재 ○○ 주가에서 가장 중요한 숫자는 ~"처럼 **핵심 숫자 한 줄 요약**으로 시작.
- 증권사 리포트 표·이미지를 캡처해 그대로 삽입(이미지 10개, 표 15개로 매우 데이터 집약적).
- 표는 "사업부문별 실적 전망" 같은 작은 단위로 여러 개 쪼개서 삽입.
- 문장은 매우 짧고(1문장 1단락), 굵은 글씨로 핵심 수치 강조.

**3) cswcul — 매크로 데이터 해설형**
- 정중한 존댓말체("~했습니다", "~로 보입니다")로 데이터를 차분히 해설.
- 통계 수치(비농업 취업자 수, 실업률 등)를 본문에 자연스럽게 풀어서 설명, 표/이미지로 시각화 보조.
- 핵심 키워드만 별도 줄에 단독 배치해 강조("영구 실직자", "완화 편향").
- 글 끝에 "[참고하면 좋은 글] ▶ ..." 형식으로 관련 글 링크 1개 추천.
- 마지막 문단에서 다음에 확인할 포인트를 던지며 글을 마무리(시리즈성 암시).

**4) hong8706 — 외신 칼럼 번역·소개형**
- 매번 동일한 인트로 템플릿: "안녕하세요~ 33년차 이코노미스트 홍춘욱입니다. (현재 직함/소속/최근 성과 업데이트)".
- 소개할 외신 기사의 제목과 원문 링크를 먼저 제시("Solar Passes Coal in Historic Shift... - Bloomberg").
- `★★★` 구분선으로 "여기부터 번역/요약 시작"을 표시.
- 이후 외신 기사를 친절한 설명체로 번역·재구성, 단락마다 빈 줄로 분리.
- 신뢰도를 높이는 출처/통계기관명을 자주 인용(엠버, EIA, SEIA 등).

**5) dysaghwls — 정책/혜택 정보형**
- 생활 밀착형 공감 멘트로 시작("사업을 하다 보면 생각보다 자주 자금 문제를...").
- 소제목을 `✅` 이모지 + 굵은 텍스트로 표시("✅ 개인신용평점 하위 20% 이하").
- 정책 변경사항을 "기존 vs 변경" 대비 구조로 설명(2,000만원→3,000만원, 6개월→2년).
- 이미지 12개로 관공서/공식 안내 캡처를 다수 첨부해 신뢰도 보강.

**6) press02 — 종목 리포트 해설형 (FACT형)**
- 도발적 질문형 제목 + 본문 첫 줄도 같은 질문 반복("목표가 90만 원, 너무 과한 숫자일까요?").
- 첫 문단에서 "오늘은 ~의 근거와 주의할 점을 숫자로 나눠보겠습니다"로 글의 구조를 미리 안내.
- `FACT` 라는 라벨을 단독으로 배치한 뒤 소제목처럼 활용("FACT. 현대차 목표가 90만 원, 숫자부터 검산합니다").
- 카드형 요약 이미지(인포그래픽 스타일)를 글 최상단에 배치.
- 표로 "검산표"를 제공해 숫자의 근거를 투명하게 공개.

**7) yminsong — 투자철학 에세이형**
- "와이민입니다"로 시작하는 친근한 1인칭 인트로.
- 본문 전체를 `1. 2. 3. ...` 번호 리스트로 구성(각 번호가 하나의 작은 문단/논지).
- 전문 용어(FOMO, Mental Hedge)를 한글 풀이와 함께 병기.
- 독자를 별칭으로 호명("와블리 여러분들의 건승을 기원합니다")해 커뮤니티 소속감 형성.
- 글 하단에 CC 라이선스 표기("저작자 명시 필수 / 영리적 사용 불가 / 내용 변경 불가").

**8) jakojako — 카페 이전형 (참고 패턴)**
- 제목만 던지고 본문은 `naver.me/...` 단축링크 카드 하나뿐 — 실제 분석글은 **비공개 네이버 카페**로 이전.
- 무료 블로그를 "맛보기/공지"용으로만 쓰고, 본 콘텐츠는 유료/회원제 커뮤니티로 유도하는
  수익화 패턴의 실제 사례. (레이아웃 참고용이 아니라 **수익화 구조 참고용**.)

**9) piersn — 종목 매매일지형**
- 시황을 먼저 1~2문단으로 요약하고, 본인의 매매 행동까지 솔직하게 공개("나는 일단 전량 익절 후...").
- `-제목-` 형태로 섹션을 구분(`-HPSP 주가 상한가 이유`, `-기업 분석 (재무제표 기준)`, `-투자 포인트`).
- 각 섹션 내부는 `① ② ③` 원문자 번호 + `.`으로 시작하는 불릿 설명.
- 종목분석 글의 4단 구성: 상한가 이유 → 기업 분석(사업구조/수익성/재무안정성/최근실적) → 투자 포인트 → (전망).

**10) onepercentkr — 자산배분 투자일기형**
- "1년만에 글을 올리네요" 같은 근황 토크로 시작 — 정기 발행이 아닌 비정기 일기 형식.
- 자산군별 포트폴리오 비중(금/은/우라늄/오일/배당주/코인)을 텍스트로 나열.
- 한두 문장마다 줄바꿈하는 네이버 특유의 짧은 호흡, 이미지 8개로 실제 자산 현황/차트 캡처 첨부.
- 매크로 전망 → 자신의 포트폴리오 변화 계획(코인 정리 시점 등) 순서로 전개.

### 레이아웃 분석

**1) hannunestock — 심층 종목분석형 (가장 무거운 포맷, 9,751자)**
- `📌 목차`로 시작해 8개 대분류 번호 헤딩(1.~8.) + 그 안에 다시 소제목.
- 구성 순서: 기업 첫인상 요약 → 기업 개요(역사/상장) → 핵심 사업 3축 → 매출 구조 →
  산업 내 위치 & 경쟁사 비교(표) → 최근 뉴스 & 단기 모멘텀 → 거시환경 → 재무 상태(2021~2025, 표).
- 최신 공시 날짜를 명시("2026년 1분기 실적 (2026.06.01 분기보고서 공시)")해 신뢰도/최신성 강조.
- 표는 경쟁사 비교, 재무 추이 등 데이터 중심 섹션에 집중 배치.

**2) gobyt — 종목 이슈 분석형**
- 1~7번 번호 헤딩으로 "회사 소개 → 주가 변동 이유 → 현재 흐름 핵심 → 기대 요인 → 리스크 →

## 부록: 선정 과정 참고사항
- 네이버 블로그는 RSS(`rss.blog.naver.com/{blogId}.xml`)로 최신 글을 확인 후 `PostView.naver`로 본문을 가져왔다.
- 티스토리는 스킨마다 본문 클래스가 달라(`article-view`, `entry-content`, `tt_article_useless_p_margin` 등)
  여러 후보를 시도해 추출했다.
- Blogspot은 `<div class='post-body ...'>` (싱글 쿼트 속성)가 표준 컨테이너였다.
- "잘나가는 대표 블로그"의 공식 순위 데이터는 존재하지 않아, 검색 노출 빈도·콘텐츠 충실도·
  최신 발행 여부를 기준으로 실용적으로 선정했다. 특히 Blogspot은 한국어 주식/재테크 콘텐츠의
  절대량이 네이버·티스토리보다 적어, 일부는 SEO 목적의 AI 생성형 블로그가 포함되어 있다
  (오히려 "AI 생성 콘텐츠 템플릿" 참고 자료로서 가치가 있다).
