## Part 5 - Repurposing 시스템

**코드베이스:**
- `my-repurpose-deploy-repo-basis.zip` (CH1-CH5 공통)

**핵심 개념:** One Source Multi Use (OSMU)
→ 하나의 콘텐츠를 여러 플랫폼에 맞게 재가공

### CH1. Repurposing 전략

#### 클립 목록
- **Clip 01** - 아이디어 1개로 5개 채널 동시 점령하기 & 환경 세팅
- **Clip 02** - 롱폼의 정의와 선택
- **Clip 03** - 플랫폼별 변환 매트릭스 & 개발 템플릿 내장 스킬 소개

**전략:**
```
1개 롱폼 콘텐츠 → 5개 플랫폼 변환
- YouTube (롱폼)
- Instagram (캐러셀/릴스)
- LinkedIn (포스트)
- 블로그 (아티클)
- 뉴스레터
```

### CH2. 글 → 글 변환

**플랫폼별 톤앤매너 자동 변환**

#### 클립 목록
- **Clip 01** - 유튜브/블로그 → Thread: 긴 글에서 핵심만 쏙 뽑아 스레드 감성으로 요약하기
- **Clip 02** - 유튜브/블로그 → 링크드인: 비즈니스 톤으로 세련되게 재가공하기
- **Clip 03** - 유튜브 → 뉴스레터: 정성껏 쓴 대본과 영상을 또 다른 롱폼으로 확산하기
- **Clip 04** - 유튜브 → 블로그: 영상 촬영용 대본을 읽기 좋은 포스팅으로 순식간에 바꾸기

**변환 매트릭스:**
| 소스 | 타겟 | 변환 포인트 |
|------|------|------------|
| YouTube 대본 | Thread | 핵심 요약, 짧은 문단 |
| YouTube 대본 | LinkedIn | 비즈니스 톤, 전문성 |
| YouTube 대본 | Newsletter | 스토리텔링, 친근함 |
| YouTube 대본 | Blog | SEO 최적화, 읽기 쉬움 |

### CH3. 글 → 이미지 변환

**텍스트 콘텐츠를 시각 자료로**

#### 클립 목록
- **(번외)** 인스타그램 이미지 로컬 다운로드 가이드
- **Clip 01** - 리서치한 카드뉴스를 바탕으로 디자인 템플릿 구성하기
- **Clip 02** - 문장 → 카드뉴스: 글 속의 핵심 문장을 디자인된 카드뉴스로 자동 전환하기
- **Clip 03** - 요약 → 그림: 복잡한 줄글을 한눈에 들어오는 인포그래픽으로 시각화하기

**활용 예시:**
- 블로그 글 → Instagram 카드뉴스
- 통계 데이터 → 인포그래픽
- 인용구 → Quote 이미지

### CH4. 영상 → 숏폼 변환

**상태:** 클립 목록 없음 (진행 중일 가능성)

**예상 내용:**
- 롱폼 영상 → YouTube Shorts
- 롱폼 영상 → Instagram Reels
- 자막 추출 및 편집

### CH5. Repurposing 워크플로우

**상태:** 클립 목록 없음 (진행 중일 가능성)

**예상 내용:**
- 전체 파이프라인 연결
- 자동화 워크플로우 구축
- n8n 또는 Make 활용
