# 수익 다각화 7갈래 — 심화 리서치

> 전제: 메인 채널(AI 영상) 외 추가 수익원 검토. 각 갈래: 정의 / 시장 현실 / 단가 / 진입장벽 / 키묘님(분석형 빌더) 적합도 / 주의점.
> 작성 2026-06-16 · 수치는 업계 공개 데이터 기반(편차 큼).

## 1. 디지털 상품 판매 (Gumroad 등)

**정의**: 프롬프트팩·n8n 워크플로우 템플릿·Notion 템플릿 등 "파일"을 만들어 반복 판매.

**시장 현실**
- AI 프롬프트팩·템플릿·자동화 워크플로우가 Gumroad 상위 카테고리. 마진 90%+, 한 번 만들면 반복 판매(재고·배송 없음).
- ⚠️ 저품질 양산이 범람 → 테스트·문서화·지원이 된 상품만 꾸준히 팔림.

**단가**
- 단품 $5 전략은 함정. **번들/볼트가 핵심** (예: 5종 묶음 "Vault" $27).
- 템플릿 $19~79. n8n 템플릿은 단순 $29 ~ 산업특화 $299+, 번들 $199~499.
- 사례: 상위 템플릿 제작자 월 $5,000~50,000+, 평범한 시스템도 월 $2,500 수준 보고.

**진입장벽**: 낮음. 단 트래픽(유입)이 필요.

**키묘님 적합도: ★★★★★ (최상)**
- n8n 템플릿·Claude 설정·프롬프트팩은 채널 만들며 **자동 축적되는 부산물**.
- passive에 가깝고, 분석·빌더 성향과 일치. 채널이 곧 트래픽 소스(시너지).

**주의**: 마케팅이 관건 → 채널이 자리잡아야 전환. 채널 선행 후 출시 권장.

## 2. 강의·클래스 (한국 플랫폼)

**정의**: "AI로 영상 만드는 법" 등을 강의로 판매.

**시장 현실**
- 플랫폼: 클래스101(구독 전환형), 인프런(개발 특화), 패스트캠퍼스(취업 연계), 콜로소·탈잉·유데미.
- ⚠️ 클래스101은 **수익의 약 40%를 플랫폼이 가져가고**, 촬영·편집·자료·요약 등 **제작 전부를 크리에이터가 부담**. 구독 모델 전환 + 강의 양산 제안으로 개별 수익이 압박받는 상황.
- 유튜브 무료 콘텐츠 범람으로 유료 전환 장벽 상승.

**단가**: 개별 판매 시 클래스당 20~30만원대(과거). 구독형은 시청 시간 기반 정산.

**진입장벽**: 제작 부담 큼. **단, 키묘님은 2번 트랙(과정/비하인드)이 곧 강의 소재** → 부담 상쇄.

**키묘님 적합도: ★★★★ (높음, 단 응대 약간)**
- "AI 영상 제작법"은 한국 수요 큼. 목소리 해설로 신뢰 쌓이면 전환 자연스러움.
- 단 수강생 Q&A 응대가 일부 발생(성향 최하위 요소).

**팁**: 플랫폼(40% 수수료) 대신 **Gumroad·자체 판매**로 강의를 팔면 마진 훨씬 높음. 플랫폼은 노출용으로만.

## 3. 제휴 (어필리에이트)

**정의**: 영상에서 쓰는 AI 툴을 소개하고 추천 링크 수수료.

**시장 현실**
- **반복 수수료(recurring)** 모델이 핵심. 예: ElevenLabs는 PartnerStack 통해 **첫 12개월간 22% 반복 수수료**(구독 유지 동안 매월 정산), 상한 없음.
- ⚠️ **Higgsfield는 2026년 6월 기준 제휴(referral) 프로그램 없음.** 'Earn'(콘텐츠 게시 보상)만 운영 → 추천 수수료와 다름.
- 제품 안 만들고 billing·지원도 없음. 청중이 이미 필요로 하는 툴을 연결.

**단가**: ElevenLabs 예시 — Creator 플랜($22/월) 20명 추천 시 월 약 $97 반복.

**진입장벽**: 매우 낮음(가입·링크). 단 청중이 있어야 효과.

**키묘님 적합도: ★★★★ (높음, 노동 적음)**
- 콘텐츠와 100% 결합, 추가 노동 거의 없음.
- 단 **채널이 선행**되어야 전환 발생.

**주의**: 한국 시청자 대상이면 **한국에서 가입·결제하는 툴** 위주로 골라야 전환됨. Higgsfield 소개는 (제휴 없으니) 수익화 불가 → 대체 툴로.

## 4. 유료 뉴스레터·멤버십 (한국)

**정의**: AI 영상·자동화 인사이트를 유료 구독으로.

**시장 현실**
- **스티비**: 플랫폼 수수료 0, **결제 수수료 3.2%만** 차감 → 마진 최고 수준. 아카이빙·구독 관리 제공.
- **메일리**: 구독 + 후원("커피") 기능. 발행 글이 블로그로 쌓여 검색 노출.
- ⚠️ 한국 시장 규모는 작음. (Substack 상위 10은 연 30억 수준이나 미국 스케일.) 무료 콘텐츠 범람으로 유료 전환 어려움.

**단가**: 월 구독료는 본인 책정(보통 월 5,000~15,000원대).

**진입장벽**: 낮음. 단 **꾸준한 글쓰기 노동**이 영상과 별개로 필요.

**키묘님 적합도: ★★★ (중)**
- 분석·정리 성향엔 맞음. 단 영상 외 추가 글쓰기 부담 → 시간(주 5~10h)과 경합.

## 5. 자동화 대행 (AI 자동화 에이전시)

**정의**: 기업·소상공인에 n8n/AI 자동화를 구축·납품.

**시장 현실**
- 수요는 큼. 린하게 특화하면 6~12개월 내 월 $10K~50K 도달 사례 보고.
- ⚠️ **냉정한 현실**: 9만+ AI 업체와 경쟁, **영업 경험 없으면 계약 매우 어려움**, 고객 언제든 해지 가능. 운영 시간의 **80%가 영업·고객관리**.

**단가**: 커스텀 구축 $1,000~10,000/프로젝트, 유지보수 $200~1,000/월, 리테이너 $2,500~5,000/월/고객.

**진입장벽**: 기술은 낮으나 **영업·고객 응대가 본체**.

**키묘님 적합도: ★ (낮음 — 성향 정면충돌)**
- 고객 직접 응대 = 성향 최하위. 영업 80% + 주 5~10h 한계와 충돌.
- 단가는 높지만 키묘님 성향엔 **비추천**.

## 6. 마이크로 SaaS / 단일 툴

**정의**: 직접 만든 좁은 기능(자막 생성기, 썸네일 A/B 등)을 구독 제품으로.

**시장 현실**
- 좁은 단일 기능 구독. 순이익 70~90%, 솔로 월 $5K~50K 사례.
- n8n 워크플로우를 제품으로 래핑 가능하나, **워크플로우 선택이 수익을 좌우**. 돈 되는 건 좁고 반복적인 것(리드 보강·문서 파싱·리포트 생성). "한 번 돌릴 것"은 스크립트지 제품 아님.

**단가**: 월 구독 $9~49 박리다매.

**진입장벽**: 중. 제작 + 검증 + 약간의 고객 지원.

**키묘님 적합도: ★★★★ (높음, 장기)**
- "채널 → 제품" 수렴 목표와 일치. 직접 만드는 재미 + 분석.
- 단 시간·검증 필요 → 채널 자리잡은 뒤 장기 카드.

## 7. 데이터·분석 도구

**정의**: 채널 운영하며 쌓는 성과/시황 데이터를 도구로.

**시장 현실**
- 마이크로 SaaS의 한 종류. 콘텐츠 분석 도구 시장은 레드오션(앞서 확인)이나, **"내가 직접 쓰는 좁은 도구"** 면 차별화 가능.
- 채널 운영 데이터가 그대로 제품 재료(씨앗 데이터).

**키묘님 적합도: ★★★★★ (성향 최적, 단 장기)**
- 데이터 분석 1순위 동기와 정확히 일치. 채널·제품 수렴의 종착점.

## 추천 순서 (성향 기준)

1. **즉시·부산물형 (채널 자리잡으면)**: ① 디지털 상품 + ③ 제휴 — 둘 다 채널의 부산물, 노동 최소.
2. **채널 성장 후**: ② 강의(자체 판매로) — 2번 트랙이 곧 강의 자산.
3. **장기·제품형**: ⑥ 마이크로 SaaS / ⑦ 데이터 도구 — "채널→제품" 수렴, 성향 최적.
4. **비추천**: ⑤ 자동화 대행 — 영업·응대 부담이 성향과 정면충돌.

> 핵심: 새 수익원을 "벌이는" 게 아니라, **채널 하나에서 가지를 치듯 파생**시키는 순서. 지금은 채널에 집중하고, 1·3번을 "나중에 이렇게 연결된다"고 염두에만.
