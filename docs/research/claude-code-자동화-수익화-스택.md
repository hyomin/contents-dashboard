# 클로드 코드(Claude Code) 자동화 수익화 서비스 스택

> 2026년 6월 기준, 클로드 코드로 자동화·수익화를 할 때 실제로 많이 조합하는 서비스 정리.
> 수익화는 크게 **(A) 직접 만든 SaaS·툴 판매** 와 **(B) 콘텐츠 자동화 파이프라인** 두 갈래로 나뉩니다.

---

## 0. 한눈에 보는 표준 조합

| 목적 | "사실상 표준" 조합 |
|------|------------------|
| SaaS / 마이크로 SaaS | Next.js + Supabase + Stripe + Vercel + Claude Code |
| 콘텐츠 자동화 | n8n + Claude/Gemini + 영상·음성 생성AI + SNS API |
| 빠른 초기 수익 | 블로그/뉴스레터 자동화 → 애드센스·제휴 |

---

## A. 직접 만든 SaaS·툴 판매 (개발 출신에게 유리)

### 1) AI 코어 (두뇌)
- **Claude Code / Claude API** — 코드 생성·아키텍처 설계의 메인 엔진
- **OpenAI API (GPT)** — 보조 모델, 멀티모델 분기
- **Gemini API** — 트렌드 수집·기획 앞단 (키묘님 기존 역할 분담 그대로)

### 2) 백엔드 · DB · 인증
- **Supabase** — DB + 인증(Auth) + 스토리지. 인디 SaaS의 1순위
- **Clerk** — 인증 전용. OAuth(구글/깃허브) 붙이기 가장 편함
- **Convex** — 실시간 데이터가 중요한 앱의 대안

### 3) 배포 · 호스팅
- **Vercel** — Next.js 프론트 + 서버리스 함수. 배포 표준
- **Railway** — 풀스택(백엔드 포함) 한 번에 올릴 때
- **Cloudflare** — DNS, 엣지 함수, 캐싱

### 4) 결제 · 수익화 (돈 받는 부분)
- **Stripe** — 구독·정기결제의 글로벌 표준 (해외 타깃이면 필수)
- **Lemon Squeezy / Polar** — 세금(VAT) 자동 처리되는 "Merchant of Record"
- **Gumroad** — 템플릿·스타터킷 같은 디지털 상품 판매 (한 번에 시작하기 쉬움)
- **토스페이먼츠 / 포트원(아임포트)** — 국내 타깃 결제

### 5) 이메일 · 알림
- **Resend** — 개발자 친화적 트랜잭션 이메일 (가입·결제 알림)
- **Slack / Telegram** — 자동 리포트·알림 봇

### 6) 분석
- **PostHog** — 제품 분석 + 세션 리플레이 (오픈소스, 셀프호스팅 가능)
- **Google Analytics** — 기본 트래픽 분석

### 7) 시간 절약용 스타터 템플릿
- **ShipFast** — 결제·인증 미리 붙은 유료 보일러플레이트
- **Next.js SaaS Starter** — 무료 시작점
- **CLAUDE.md 템플릿 / Config Kit** — 클로드 코드가 매번 일관된 코드를 뽑게 만드는 설정 파일 (토큰 절약 효과도 큼)

> 💡 *비유*: SaaS 만들기는 "조립 PC"와 같습니다. CPU(Claude), 메인보드(Next.js), 저장장치(Supabase), 전원공급(Vercel), 결제 단말기(Stripe)를 끼우면 완성품이 됩니다.

## B. 콘텐츠 자동화 파이프라인 (키묘님 핵심 프로젝트)

### 1) 워크플로우 자동화 (전체 흐름의 중심)
- **n8n** — 오픈소스, 셀프호스팅 가능, 무제한 실행. 자동화의 심장
- **Make** — 노코드, n8n보다 UI 직관적
- **Zapier** — 가장 쉬움. 다만 실행 횟수 과금이 빨리 참

### 2) 기획 · 스크립트 (글 단계)
- **Gemini** — 트렌드·소재 수집, 기획 앞단
- **Claude** — 스크립트 작성·구조화 (키묘님 기존 분담 유지)
- **Obsidian** — 소재·기획 아카이브 (원소스 보관함)

### 3) 영상 · 이미지 · 음성 생성 (제작 단계)
- **Higgsfield** — AI 영상 생성
- **ElevenLabs** — AI 음성(나레이션, 시니어 타깃에 또렷한 TTS)
- **이미지 생성** — Gemini(나노바나나)·미드저니 등 썸네일/장면용

### 4) 배포 · 유통 (원소스 멀티유즈)
- **YouTube Data API** — 쇼츠·롱폼 업로드 자동화
- **Instagram / Meta API** — 릴스 자동 게시
- **블로그(워드프레스/티스토리 API)** — 파생 글 자동 발행 (초기 수익 1순위)

### 5) 수익화 출구
- **Google AdSense** — 블로그·유튜브 광고
- **제휴 마케팅(쿠팡파트너스 등)** — 콘텐츠 내 링크
- **유료 멤버십 / 디지털 상품** — 채널 성장 후

> 💡 *비유*: 콘텐츠 자동화는 "컨베이어 벨트 공장"입니다. n8n이 벨트, 제미나이·클로드가 설계실, 영상·음성AI가 가공 기계, SNS API가 출고장입니다. 한 번 원료(소재)를 넣으면 여러 제품(쇼츠·릴스·블로그)이 동시에 나옵니다.

## C. 자주 쓰는 수익화 패턴 (참고)

1. **마이크로 SaaS** — 작은 문제 하나를 푸는 구독 툴 (Stripe 결제)
2. **디지털 상품 판매** — CLAUDE.md 템플릿·스타터킷·프롬프트팩 (Gumroad)
3. **콘텐츠 채널** — 시니어 타깃 유튜브 + 블로그 파생 (AdSense·제휴)
4. **업무 자동화 대행** — n8n 워크플로우를 클라이언트에게 구축·납품
