## 주의사항

```bash
# YouTube Data API v3 활용
curl "https://www.googleapis.com/youtube/v3/channels\
?part=id,snippet\
&forHandle=@syukaworld\
&key=YOUR_API_KEY"
```

또는 대시보드 내 YouTube Data API 연동 시 `channels?forHandle=@핸들명` 엔드포인트 활용
